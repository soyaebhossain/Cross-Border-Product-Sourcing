# Welcome — Cross-Border Product Sourcing

Welcome to the Cross-Border Product Sourcing project. This repository contains a full-stack app (Django backend + Next.js frontend) and several developer tools and scripts.

Quick links
- Repository root: ./
- Backend: ./backend/
- Frontend: ./apps/web-next/
- Local welcome page: public/welcome.html

Quick start (Windows)

1. Backend (Python/Django)

```powershell
cd backend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

2. Frontend (Next.js)

```powershell
cd apps\web-next
npm install
npm run dev
```

If you need help getting started, see the main [README](README.md).

Thanks for joining the project — enjoy exploring!
